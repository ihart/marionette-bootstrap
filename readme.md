# Marionette Bootstrap


## What is this?

A basic bootstrap using Backbone Marionette for component-based web apps.


## Local set-up

Install front-end modules:
```
npm install
```

Build it:
```
gulp
```


## Running

Run the server:
```
gulp server
```

Build the front-end and watch for changes:
```
gulp watch
```

Visit: [http://0.0.0.0:8000/](http://0.0.0.0:8000/)
